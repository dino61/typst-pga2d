This repo contains a **Tpst** code library for implementing Plane-based Geometric Algebra (**PGA**) in 2D and 3D.
See: https://bivector.net and: https://typst.app/docs/

I'm writing a document using Typst that will be an elementary introduction to Geometric Algebra assuming the
reader has virtually no knowledge of mathematics except for a casual ability to do a few of Euclid's simple ruler and compass constructions from his book "The Elements" and some basic arithmetic.
This is usually taught in elementary school at some point.
The main thesis of the document is that algebra should be viewed as a written form of geometry in the same way that
western musical notation is a written form of music.
I'm using the PGA code from this repo to generate the figures in the document about PGA.

The pga code is automatically generated using Steven De Keninck's wonderful ganja.js Javascrit library. The code includes a function called 'runTests()' to test the main functions of the library.
