Tpst code library for implementing Plane-based Geometric Algebra (PGA) in 2D and 3D.
See: https://bivector.net
See: https://typst.app/docs/

I'm writing a document using Typst that will be an elementary introduction to Geometric Algebra assuming the
reader has virtually no knowledge of mathematics except for a casual ability to do a few of Euclid's simple ruler and compass constructions from his book "The Elements" and some basic arithmetic. This is usually taught in elementary school at some point.
I'm using the PGA code from this repo to generate the figures in the document about PGA.
